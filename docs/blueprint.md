# **App Name**: Retos Graduados

## Core Features:

- Live Participant Onboarding: Assistant registration with QR, photo capture via camera/gallery, and role selection (Participant or Voter).
- Firestore Event Synchronization: Real-time state management using Firestore to sync screens across projectors, smartphones, and the conductor dashboard.
- Dynamic Pair Sorter: Automated matching engine that pairs active participants for duels and assigns a challenge dynamically.
- Live Anonymous Voting: Synchronized voting system where attendees vote via their devices with results hidden on the public projector.
- AI Dynamic Generator Tool: An AI-powered tool for the conductor to refine challenge instructions or generate new themed dynamics based on previous inputs.
- Tournament Flow Orchestration: Single-button management for duels, manual tie-breakers, winner confirmation, and automatic status updates.
- Visual Projector Layout: A high-contrast 16:9 optimized dashboard showing QR codes, live match-ups, and dramatic winner reveal states.

## Style Guidelines:

- Primary Color: Elegant Gold (#D4AF37) used for accents, victory states, and primary buttons to evoke a competition-winner aesthetic.
- Background Color: Night Blue (#0A111F), a deeply desaturated version of the navy primary, ensuring depth and readability on high-brightness projectors.
- Accent Color: Intense Amber (#FF7A00), an analogous hue used for interactive alerts, timers, and active vote calls-to-action.
- Headline and body font: 'Poppins' for a contemporary, geometric, and fashionable look that remains legible from long distances on a projector.
- Sleek, minimalist outline icons for competition status, featuring circular profile images to emphasize the personal nature of the event.
- Full-width card-based layouts with heavy border radius for a modern app feel, optimized for a horizontal 16:9 projector and vertical mobile interactions.
- Subtle slide-and-fade transitions when switching between waiting and competition states to provide visual feedback during high-tension reveals.
